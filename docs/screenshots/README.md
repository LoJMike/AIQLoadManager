## Capture guidelines

- Window size: 1440×900 (or 1280×800)
- Resolution: 2× retina if available (PNG)
- Set up realistic data first — no empty states
- Crop to the app window only (no desktop chrome)
- Format: PNG (not JPEG — crisp UI edges)
