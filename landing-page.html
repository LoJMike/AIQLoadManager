What it does

Stop juggling APIs.
Start shipping.

AIQ Load Manager is not a chat app. It's a background queue engine — fire off prompts and get on with your work.

⚡

Background queue processing

Prompts queue instantly and process in the background every 1.5–3 seconds. Rate limits are tracked in real time — nothing gets dropped or rejected.

🔀

Six intelligent routing modes

Auto, Balance, Cheapest, Fastest, Free Tier, and Manual modes direct each prompt to the right provider automatically. No manual switching.

💰

Cost & token tracking

See exactly how many tokens and dollars you've spent per provider, per model, over any time period. Set budget caps to avoid surprises.

🗂️

Projects & persistent history

Organize prompts into separate projects, each with its own conversation thread. History is written to local SQLite on every turn — conversations survive app restarts exactly where you left off.

🏷️

Prompt type tags

Label each prompt with visual chip tags — Chat, Research, Code, Writing, Analysis, Image, Translate, or ⚡ Urgent. Tags drive provider selection automatically and, on Pro, lift your prompt's place in the queue. The special 🌐 Web Search tag also triggers live result injection before the AI call.

💡

Live cost & token estimation

See a live token count as you type and a ranked per-provider cost table before you hit queue. Know exactly what each prompt will cost on Claude vs. Groq vs. Gemini — before you send it.

⚖️

Compare mode Pro

Send the same prompt to multiple providers simultaneously. Responses arrive in parallel and display as side-by-side columns — provider name, model, full text, copy button, token count. Instantly see whether Claude, GPT-4o, or Gemini does better on your task.

🌐

Real-time web search

Tag any prompt 🌐 Web Search and AIQ fetches live results before sending — then injects them into the system prompt. Works with every model, including fully local ones. No tool-calling support needed. Choose Tavily (1,000 free searches/month) or self-hosted SearXNG.

🔒

100% local & private

Everything stays on your machine. API keys are stored in your OS keychain. No cloud sync, no telemetry, no middleman reading your prompts.

🖥️

Native desktop app

Built with Electron and React. Runs natively on Windows 10/11 and macOS. Installs as a proper desktop app — no browser tab to lose.

FAQ

Frequently asked questions

Do I need to provide my own AI API keys?

Yes. AIQ connects to the providers you already pay for — it doesn't mark up or resell AI access. You pay providers directly at their published rates. The app helps you use those APIs more efficiently.

Monthly vs. lifetime — which should I choose?

Monthly is lower risk if you want to try before committing long-term. Lifetime typically breaks even versus monthly in 6–7 months and covers all future updates within that tier indefinitely.

Is the Free tier genuinely usable?

Yes, and there are two ways to use it at zero cost. First, install a local AI server — Ollama, LM Studio, or Jan.ai are the easiest. All run AI models on your own machine with no API key and no internet required. Second, use cloud free tiers from Gemini, Groq, or Mistral — all offer permanent free access without a credit card. The Free tier is not a trial, it's a permanent plan.

Is my data safe? Does anything leave my computer?

Your prompts go directly from your machine to the provider's API — nothing passes through our servers. API keys are stored in your OS credential store. There is no telemetry and no usage tracking on our end.

How is this different from OpenRouter or LiteLLM?

OpenRouter and LiteLLM are API gateways for developers — they require integrating an API into your own code. AIQ is a desktop GUI for humans. You don't write any code. You queue a prompt, pick a routing mode, and read the result.

When will the iOS and Android apps be available?

Mobile is currently in development. Android will arrive first given more permissive background processing. iOS follows. Both will be included at no extra charge for Starter and Pro users.

What are prompt type tags, and do I have to use them?

Tags are optional visual labels you apply to a prompt before queuing — Chat, Research, Code, Web Search, Writing, Analysis, Image, Translate, or ⚡ Urgent. They do two things: the first content tag tells the router what kind of task it is so it can prefer the right providers (e.g. Code routes toward Claude and Mistral). The ⚡ Urgent tag bumps the item's position in the queue immediately — available on all plans. On Pro, every tag type adds a priority weight, so Research and Code prompts automatically outpace untagged items.

How accurate is the cost estimate shown before queuing?

It uses the formula (inputTokens / 1M × inputRate) + (maxTokens / 1M × outputRate) with a rough token approximation of one token per four characters (±15% for English text). It shows the cheapest available model per provider so you see the best-case cost. The estimate is for orientation — actual charges from providers may vary slightly depending on system prompts and conversation history.

How does the 🌐 Web Search tag work?

Tag any prompt with Web Search before queuing. Just before the AI call, AIQ runs a live search using either Tavily (cloud, 1,000 free searches/month — sign up at app.tavily.com) or SearXNG (self-hosted Docker, completely free and private). The top 5 results are formatted into a context block and prepended to the system prompt, so the AI sees up-to-date information without needing tool-calling or function support. This works with every provider, including fully local models like Ollama and LM Studio.

Can I upgrade or downgrade later?

Monthly subscribers can switch tiers at any time — you're only billed the difference going forward. Lifetime tier upgrades are the price difference between the two tiers, paid once.

What platforms does the desktop app support?

Windows 10 and 11 (x64), and macOS (Intel + Apple Silicon). Both get identical features. Linux builds are not currently offered but are on the longer-term roadmap.
